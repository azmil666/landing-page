# GDG Landing Page

Landing page concept for a Google Developer community.

## Tech Stack
- React (Vite + TypeScript)
- Tailwind CSS
- Framer Motion

## Setup, Run & Build

Clone the repository:
```bash
git clone https://github.com/azmil666/landing-page.git
cd landing-page
````

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
